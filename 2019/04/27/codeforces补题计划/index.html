C. Vus the Cossack and Strings

题意

给01字符串a,ba, ba,b，∣a∣>=∣b∣|a| >= |b|∣a∣>=∣b∣（aaa的长度大于等于bbb的长度），求a中长度和bbb长度相等的子串不同字符数为偶数的串的个数。

样例

题解

经观察，可以得到这样的性质——串x, y的不同字符串个数的奇偶性和字符1（或0）个数的奇偶性一致。 证明:

定义f(x, i)为串x第i位的字符；N为串x, y中字符为1的个数；M为x, y字符不同的个数；isEven(N)代表N的奇偶性；
如果f(x, i) == f(y, i) == ‘0’，那么对N和M的奇偶性没有贡献；
如果f(x, i) == f(y, i) == ‘1’，那么由于isEven(N-2) == isEven(N)，对N和M的奇偶性没有贡献；
如果f(x, i)f(y, i) == ‘01’ or f(x, i)f(y, i) == ‘10’，那么改位对N和M的值都有贡献，由于isEven(N) == isEven(M) => isEven(N-1) == isEven(M-1)，所以改位对N和M的奇偶性贡献相同；
综上，N和M的奇偶性一致。
所以对于aaa我们求出1个数的前缀和再与bbb中1的个数相加即可得到不同字符个数的奇偶。

D. Vus the Cossack and Numbers

题意

有和为0的浮点数序列ai...an，输出和为0的整数序列bi...bn，其中bi == floor(ai) or bi == ceil(ai)。

样例

题解

将序列ai...an中的正数和负数向0取整后分别求和为p_sum, n_sum，可得其差值，如果正数多那么选取相应个数的负数向下取整，反之亦然。
